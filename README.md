# Trans Jogja Web Application

Aplikasi berbasis web untuk mempermudah akses informasi rute, pemantauan lokasi halte dan bus, serta layanan transaksi bagi penumpang Trans Jogja.

## Fitur Utama

- **Peta & Rute**: Visualisasi rute bus menggunakan Google Maps API dan data GeoJSON (contoh: Rute 14).
- **Pelacakan Armada**: Memantau lokasi bus secara langsung.
- **Sistem Transaksi**: Pencatatan layanan tiket dan riwayat perjalanan.
- **Manajemen Akun**: Registrasi dan autentikasi terpusat menggunakan Firebase.
- **Analisis Data**: Pengumpulan data *demand* (permintaan) dan fitur ekspor laporan ke format CSV.

## Stack Teknologi

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Backend / Database**: Firebase
- **Maps**: Google Maps JavaScript API
